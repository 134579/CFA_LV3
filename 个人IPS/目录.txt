5. return objective RRTTLLu, 重点
	1. open ended: 开放式, 难
		1. 题型1
			TIA: total investable asset: 期初可投资资产, 可以直接变卖成现金
				不包括: 自住房子, 未来能拿到的遗产
			步骤: 先算未来现金流, 再算TIA, 再算required rate of return 
			关键概念:
				maintain the real value: 跑赢通胀
			例子1:
				t=0 TIA = 100w, 
				t=1 cash inflow = 2w, outflow = 10w,
				inflation = 3%,  need to maintain the real value, 求required return
			解:
				100(1+r) +2 - 10 = 100 *(1+3%)
					=> r = 3% + (10-2/)/100 = 11%
					
				
				
			计算过程:
					现金流
						TIA = T0 cash inflow - T0 cash outflow					T = 1	
							T0 inflow: 											T0 inflow
								salary, 											salary(t=1) = salary(t=0) * (1+s), s为工资的增长速度
								inheritance, 										0
								current portfolio ...								0
							T0 outflow: 										T1 outflow
								living expense										living expense(t=1) = living expense(t=0) * (1+inflation rate)
								down payment, 首付									mortgage, 月供
								education cost
								
					公式:
						required return = (t=1 cash flow needs)/TIA + inflation rate 
							注: 这是税后收益(after tax rate of return)
						pre tax return= required return / (1- tax rate)
							计算:	TIA(1+r(1-tax rate)) = TIA(1+inflation rate) + cash flow needs
										r = (cf needs/TIA + inflation rate)/(1-tax rate)
										  = required return after tax /(1 + tax rate)
										  
						
						注:
							都是nominal return
												
					
				重要假设:
					1. 年中发生的salary,expense, 统一设置发生在年末, 不然没法算了
					2. 原来portfolio里的资产可以无成本变现,  无损变为TIA
					3. TIA(T=0)一定是税后的(注意工资, 遗产)
				
		
		2. open ended 题型2
			客户提前一年找到jjjj, t=0
			jjjj开始管理客户资产, t=1
			满足cash flow needs, t=2
			t = 1
			TIA = cash inflow - cash outflow
				inflow =
					t=0 portfolio + t=[0-1] divdend or coupon
		
	2. closed ended: 简单
		例子: 30岁有TIA, 已知退休后需要3000w, 30-退休, 每年消费10w,
		求30-退休之前, 每年的要求回报率
	
6. risk objective
	1. 两个相关
		ability
			above average, 一般是这个, 因为能找jjjj管理财产的一般有钱...
			average
			below average
			
		willingness
			above
			average
			below
	
							willingness
					above			below
		ability		
		above		above			below
		below		below			below
		只有当ability和willingness都是above average, risk tolerance才是above average
		risk tolerance = ability && willingness
	2. 影响ability因素:
		time horizen: 投资期限, ↑
		portfolio size: ↑
		required return: ↓
		spending needs: ↓
		goal importance: 目标越重要, 风险承担能力越低↓
			对比:
				living expense(重要): risk ability低
				donation(不重要): risk ability高
		flexibility: ↑
	3. 影响willingness因素:
		过去投资:  past investment style
			国债: 低
			小盘股: 高
		表现:
			过度anxiety, 低
		自己说自己不算, 就算自己说风险willingness高, 也有可能实际willingness低
